## 如何使用

### 在 TUI 中切换会话

**方法 1：查看所有会话**
```
/sessions
```

**方法 2：切换到指定会话（通过标签）**
```
/session "独立工作会话"
```

**方法 3：切换到指定会话（通过 ID）**
```
/session switch caec1b9a-be3b-489a-8dc9-76ae268f86cb
```

---

## 在主会话中与子会话通信

**给子会话发送消息：**
```
"给会话 caec1b9a-be3b-489a-8dc9-76ae268f86cb 发消息：你的任务内容"
```

或使用标签：
```
"给 TUI 独立会话 2 发消息：你的任务内容"
```

---

## 创建新会话

**在主会话中说：**
```
"创建一个新会话"
```

我会创建新会话并更新这个文件。
